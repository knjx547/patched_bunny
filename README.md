# !SETUP
first add your discord bot token to ```bot_token```
then your ```webhook_url``` for infodump
and then your user id or if you want more add more.

# !commands
```
admin - check for admin privileges
cmd - command prompt
website - open a website
wallpaper - change wallpaper
blockinput - block keyboard and mouse
unblockinput - unblock keyboard and mouse
open - open an app
download - download a file from pc
upload - upload a file onto pc
screenshot - take a screensoht
sound - play only a .wav sound
msg - pop up msg
kill - kill an open app
bluescreen - bluescreen pc
shutdown - shutdown pc
restart - restart pc
cam - take a pic from webcam
tts - text to speech
persis - glue rat to system
infodump - steal info and discord token
startkeylog - starts a keylogger
dumpkeylog - dumps the keylogger
stopkeylog - stops the keylogger
clskeylog - tries to clear any trace of keylogger
spread - tries to spread itself
disabletaskmgr - disable task manager
enabletaskmgr - enable task manager
clipboard - paste last thing copied
displayoff - turn off display
displayon - turn on display
sleep - put pc to sleep
logout - log user out of pc
```
